Scraping Blog
==================================================

what this?
--------------------------------------------------
This is an aggregate web site.
it's developed modulable, and can be expanded by scraping script module.

require
--------------------------------------------------
* node v0.6.11 (tested)
* npm  v1.1.1  (tested)

usage
--------------------------------------------------
1. get source
    git clone git://github.com/darth-hal/scrape.git
2. resolve dependencies
    npm install -d
3. run application
    node app.js
